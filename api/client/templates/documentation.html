Section 1

Setup

Welcome to the setup! First you will need to UnZip the file. You will then locate BitPGP.EXE, once you locate it you need to double click to run the application. Once you run the application a console will appear. Congradulations, you have setup BitPGP. [NOTE]: DO NOT REMOVE `accounts.bencacc` it is an essential file.

Section 2

Account Section

Create an Account

To create an account you need to firstly run the application. Once you run the application you will enter the number `1` into the console, and click enter. Once you do this you will see a menu listing more options, enter the number `3` to create an account. You will then enter the username into the console and click enter once more.

Delete an Account

To delete an account you will need to run the application. Once you run the application you will enter the number `1` into the console, and then click enter. Once you have done this you will then see a menu and you will now need to enter the number `2`, and type an accounts name to delete.

Viewing Keys

To view your accounts, and their keys you will run the application. Once you have ran the application you will enter the number `1` into the console and then click enter. Once you have done this step you will then enter the number `4` into the console and click enter, you will then see the list of your accounts and their keys.

Section 3

Encrypting

Encrypting a message.

Encrypting plays a big role in BitPGP, you will need to run the application. Once you run the application you will then need to enter encryption mode, once you do that you will then need to enter the account name you want to use [example: John, Bill, etc], then you will enter the message to encrypt. Once you do this you will need to send the recipent your account KEY, and the encrypted message.

Section 4

Decrypting

Decrypting a message.

Decrypting a message is important to actually see it, you will need to run the application. Once you run the application you will then need to enter decryption mode, once you do that you need to ask for the persons key who is giving you the message. Once you do that you will enter the key into the prompt, and then the message. It should return the original message, hopefully.
